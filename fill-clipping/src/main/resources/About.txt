Программа демонстрирует работу трех алгоритмов для построения примитивов графики на основе алгоритма Брезенхема для отрезка и окружности. Слева рисуют самодельные алгоритмы. Код с алгоритмами находится в пакете com.nocmok.opengl.fillclip.drawer. Справа рисуют алгоритмы из java.awt.Graphics2D https://docs.oracle.com/en/java/javase/15/docs/api/java.desktop/java/awt/Graphics2D.html

Программа написана на языке java 11 с использованием библиотеки для построения клиентских приложений javafx 17. Для сборки использовался gradle 7.2. Программа разрабатывалась в IntelliJ IDEA Community
