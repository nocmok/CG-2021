Программа демонстрирует работу 1) Алгоритма построчной заливки, 2) Алгоритма заливки полигона, 3) Алгоритма отсечения прямоугольником Сазерленда-Коэна. Каждая 1),2),3) демо находится на своей вкладке. Переключение между вкладками - по нажатию на кнопки в левом нижнем углу.

В демо 1) можно рисовать фигуры, которые выбираются по кнопкам в левом верхнем углу. Для заливки нужно нажать на кнопку "fill" и нажать на экране на место затравочной точки

В демо 2) нужно задать полигон по точкам. Точки добавляются нажатием на экран. Точки рассматриваются в порядке добавления. Между двумя соседними точками добавляется ребро. Между первой и последней точкой добавляется ребро. Заданный таким образом полигон заливается и выводится на экран.

В демо 3) есть два экрана. Левый экран - отображает исходные данные. На левый экран можно добавлять отрезки и прямоугольники.Выбор фигуры - по кнопкам в левом верхнем углу. Можно добавлять сколько угодно отрезков. Прямоугольник задает область отсечения, поэтому он всегда один. При добавлении нового прямоугольника старый удаляется. Порядок добавления фигур не имеет значения. Отрезки добавленные на левый экран подсвечиваются разными цветами в зависимости от их видимости. Зеленый - отрезки, которые целиком лежат в окне. Красный - отрезки которые целиком лежат в одной из сторон относительно окна. Оранжевый - остальные отрезки. На правом экране - результат отсечения алгоритмом Сазерленда-Коэна - прямоугольная область и фрагменты исходных отрезков, которые попали в окно.

Все демо скроллятся (тачпад или колесико/shift + колесико), зумятся и очищаются по нажатию на соответствующие кнопки наверху экрана.

Программа написана на языке java 11 с использованием библиотеки для построения клиентских приложений javafx 17. Для сборки использовался gradle 7.2. Программа разрабатывалась в IntelliJ IDEA Community
