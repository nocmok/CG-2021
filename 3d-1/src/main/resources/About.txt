В программе есть 4 3d-сцены. Сцены отличаются только моделью, которая помещена на сцену. В правой части сцены панель, которая позволяет регулировать следующие параметры:
1) Сдвиг объекта по осям x,y,z (Блок "Shift")
2) Поворот объекта вокруг осей x,y,z (Блок "Rotation")
3) Масштабирование объекта вдоль осей x,y,z по отдельности, или вдоль всех осей сразу (Блок "Scaling")
4) Проекция (Блок "Projection"). Доступна параллельная и центральная проекция. Для центральной проекции можно задать положение точки схода (Блок "pov").
5) Порядок применения операций сдвига и вращения (Блок "Order"). Первая опция - сначала сдвиг, потом вращение. Вторая опция - наоборот

Программа написана на языке java 11 с использованием библиотеки для построения клиентских приложений javafx 17. Для сборки использовался gradle 7.2. Программа разрабатывалась в IntelliJ IDEA Community
