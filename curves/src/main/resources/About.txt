Программа демонстрирует построения сплайнов и составных кривых. Для каждой кривой есть своя сцена, которую можно выбрать кнопками снизу. Для того чтобы отрисовать кривую, нужно разместить на сцене опорные точки с помощью мыши. Чтобы стереть кривую на каждой сцене есть кнопка clear. Составных кривые также можно замкнуть. Для этого справа от кнопки clear есть чекбокс.

Программа написана на языке java 11 с использованием библиотеки для построения клиентских приложений javafx 17. Для сборки использовался gradle 7.2. Программа разрабатывалась в IntelliJ IDEA Community
